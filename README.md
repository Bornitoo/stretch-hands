# Stretch Hands 🖐️➰

Растягивай руку как **резиновую** прямо в браузере через вебкамеру. Резкий бросок — удар
со вспышкой и звуком. Два режима рендера: мультяшная резиновая конечность и реалистичный
WebGL-лизквифай твоего настоящего предплечья.

Всё работает **локально в браузере** — видео и записи никуда не отправляются.

> Оригинальный фан-проект, вдохновлён эффектами «растягивающейся руки». Без отсылок к
> чьим-либо товарным знакам.

## Что внутри

- **Трекинг:** MediaPipe `HandLandmarker` (21 точка кисти) + `PoseLandmarker` (плечо/локоть
  как «якорь» руки). Всё на устройстве, через `@mediapipe/tasks-vision`.
- **Рендер:**
  - *Мультяшный* — гладкая резиновая конечность на canvas2D с «желейной» физикой.
  - *Реалистичный* — WebGL: реальные пиксели предплечья растягиваются вдоль удлинённой оси.
- **Фишки:** «гам-гам пистолет» (бросок = удар, импакт, тряска, звук; жесты кулак/ладонь),
  снимок PNG и запись видео (webm) с эффектом.

## Структура

```
public/
  index.html, css/style.css
  js/
    tracker.js          камера + Hand + Pose → единый объект кадра
    stretch.js          ЧИСТАЯ логика эффекта (геометрия, физика, жесты) — тестируется в Node
    render-cartoon.js   рендер №1 (canvas2D)
    render-realistic.js рендер №2 (WebGL-лизквифай)
    pistol.js           удар/откат/импакт
    sfx.js              звук на Web Audio
    capture.js          снимок + запись
    app.js              сборка и UI
  vendor/mediapipe/     рантайм и модели (не в git — см. ниже)
test/stretch.test.mjs   юнит-тесты ядра
wrangler.toml           деплой на Cloudflare Workers (Static Assets)
scripts/fetch-models.sh загрузка моделей MediaPipe
```

## Запуск локально

```bash
bash scripts/fetch-models.sh     # один раз: скачать рантайм и модели в public/vendor/mediapipe
npx wrangler dev                 # http://localhost:8787
```

Открой в Chrome, разреши доступ к камере, жми «Включить камеру».
`?debug=1` — каркас руки и панель настроек. `?` без камеры эффект не запускается.

## Тесты

```bash
node --test test/*.test.mjs
```

## Деплой (Cloudflare Workers)

```bash
export CLOUDFLARE_API_TOKEN=...   # токен с правом Workers
export CLOUDFLARE_ACCOUNT_ID=...
npx wrangler deploy
```

## Приватность

Инференс, рендер и запись — целиком в браузере. Никакие кадры не уходят на сервер.
